---
title: "getNftEvents"
description: "Returns transactions for an NFT collection across any marketplace(s)."
---

**Returns:** [NftEventsConnection](/api-reference/types/nfteventsconnection)

### Arguments

<ResponseField name="address" type="String">
  The contract address of the NFT collection.
</ResponseField>

<ResponseField name="exchangeAddress" type="String">
  The NFT marketplace address to filter by.
</ResponseField>

<ResponseField name="poolAddress" type="String">
  The NFT pool address to filter by.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the NFT collection is deployed on.
</ResponseField>

<ResponseField name="tokenId" type="String">
  The token ID to filter by.
</ResponseField>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>

<a href="/api-reference/types/eventquerytimestampinput">
  <ResponseField name="timestamp" type="EventQueryTimestampInput">
    The time range to filter by.
  </ResponseField>
</a>

<ResponseField name="limit" type="Int">
  The maximum number of NFT events to return.
</ResponseField>

<ResponseField name="includeTransfers" type="Boolean">
  No description provided
</ResponseField>

### Schema


```graphql 
"""Returns transactions for an NFT collection across any marketplace(s)."""
getNftEvents(address: String, exchangeAddress: String, poolAddress: String, networkId: Int!, tokenId: String, cursor: String, timestamp: EventQueryTimestampInput, limit: Int, includeTransfers: Boolean): NftEventsConnection
```
