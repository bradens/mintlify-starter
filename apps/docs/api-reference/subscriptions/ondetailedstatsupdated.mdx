**Returns:** [DetailedStats](/api-reference/types/detailedstats)

### Arguments

<ResponseField name="pairId" type="String">
  The ID of the pair (`address`:`networkId`).
</ResponseField>

  </ResponseField>
</a>

### Schema


```graphql 
"""Live-streamed bucketed stats for a given token within a pair."""
onDetailedStatsUpdated(pairId: String, tokenOfInterest: TokenOfInterest): DetailedStats
```
