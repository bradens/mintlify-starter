---
title: "liquidityLocks"
description: "Returns liquidity locks for a given pair."
---

**Type:** Query

**Returns:** [LiquidityLockConnection](/api-reference/types/liquiditylockconnection)

## Arguments for liquidityLocks

| Name | Description |
|------|-------------|
| `pairAddress` (String) | The address of the pair. |
| `tokenAddress` (String) | The address of the token. |
| `networkId` ([Int!](/api-reference/types/int)) | The network id of the pair. |
| `cursor` (String) | A cursor for use in pagination. |
