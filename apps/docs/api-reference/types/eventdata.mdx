### Union Types

This union can be one of the following types:

- [BurnEventData](/api-reference/types/burneventdata)
- [MintEventData](/api-reference/types/minteventdata)
- [SwapEventData](/api-reference/types/swapeventdata)
- [PoolBalanceChangedEventData](/api-reference/types/poolbalancechangedeventdata)

### Schema


```graphql 
"""Event-specific data for a token transaction."""
union EventData = BurnEventData | MintEventData | SwapEventData | PoolBalanceChangedEventData
```
