### Union Types

This union can be one of the following types:

- [PriceEventWebhookCondition](/api-reference/types/priceeventwebhookcondition)
- [MarketCapEventWebhookCondition](/api-reference/types/marketcapeventwebhookcondition)
- [TokenPairEventWebhookCondition](/api-reference/types/tokenpaireventwebhookcondition)
- [NftEventWebhookCondition](/api-reference/types/nfteventwebhookcondition)
- [RawTransactionWebhookCondition](/api-reference/types/rawtransactionwebhookcondition)


```graphql WebhookCondition
"""Webhook conditions that must be met for each webhook type."""
union WebhookCondition = PriceEventWebhookCondition | MarketCapEventWebhookCondition | TokenPairEventWebhookCondition | NftEventWebhookCondition | RawTransactionWebhookCondition
```
