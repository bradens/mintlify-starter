---
title: "LatestToken"
description: "Metadata for a newly created token."
---

**Type:** [LatestToken](/api-reference/types/latesttoken)

## Fields

| Name | Description |
|------|-------------|
| `id` → [String!](/api-reference/types/string) | The id of the new token. (tokenAddress:networkId) |
| `tokenAddress` → [String!](/api-reference/types/string) | The contract address of the new token. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the token is deployed on. |
| `blockNumber` → [Int!](/api-reference/types/int) | The block number of the token contract&#39;s creation. |
| `transactionIndex` → [Int!](/api-reference/types/int) | The index of the transaction within the block. |
| `traceIndex` → [Int!](/api-reference/types/int) | The index of the trace within the token contract&#39;s creation transaction. |
| `transactionHash` → [String!](/api-reference/types/string) | The unique hash for the token contract&#39;s creation transaction. |
| `blockHash` → [String!](/api-reference/types/string) | The unique hash for the token contract&#39;s creation block. |
| `timeCreated` → [Int!](/api-reference/types/int) | The unix timestamp for the creation of the token. |
| `creatorAddress` → [String!](/api-reference/types/string) | The address of the token creator. |
| `creatorBalance` → [String!](/api-reference/types/string) | The token creator&#39;s network token balance. |
| `tokenName` → [String!](/api-reference/types/string) | The name of the token. |
| `totalSupply` → [String!](/api-reference/types/string) | The total supply of the token. |
| `tokenSymbol` → [String!](/api-reference/types/string) | The symbol of the token. |
| `decimals` → [Int!](/api-reference/types/int) | The token&#39;s number of decimals. |
| `simulationResults` → [[LatestTokenSimResults!]!](/api-reference/types/latesttokensimresults) | Simulated token contract results, if available. |
