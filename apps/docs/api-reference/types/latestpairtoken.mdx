### Fields

<ResponseField name="address" type="String!" required>
  The contract address for the token.
</ResponseField>

<ResponseField name="currentPoolAmount" type="String!" required>
  The amount of `token` currently in the pair.
</ResponseField>

<ResponseField name="decimals" type="Int!" required>
  The precision to which the token can be divided. For example, the smallest unit for USDC is 0.000001 (6 decimals).
</ResponseField>

<ResponseField name="id" type="String!" required>
  The ID of the token (`address:networkId`).
</ResponseField>

<ResponseField name="initialPoolAmount" type="String!" required>
  The initial amount of `token` added to the pair.
</ResponseField>

<ResponseField name="name" type="String!" required>
  The name of the token.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the token is deployed on.
</ResponseField>

<ResponseField name="pairId" type="String!" required>
  The ID of the pair (`pairAddress:networkId`).
</ResponseField>

<ResponseField name="poolVariation" type="Float!" required>
  The percent change `token` remaining in the pair since the initial add.
</ResponseField>

<ResponseField name="symbol" type="String!" required>
  The symbol for the token.
</ResponseField>
