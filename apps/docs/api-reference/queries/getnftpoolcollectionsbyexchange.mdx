---
title: "getNftPoolCollectionsByExchange"
description: "Returns an NFT collection with pool stats for a given AMM NFT marketplace."
---

**Type:** Query

**Returns:** [GetNftPoolCollectionsResponse](/api-reference/types/getnftpoolcollectionsresponse)

## Arguments for getNftPoolCollectionsByExchange

| Name | Description |
|------|-------------|
| `exchangeAddress` ([String!](/api-reference/types/string)) | The NFT AMM marketplace address to filter by. |
| `networkId` ([Int!](/api-reference/types/int)) | The network ID the NFT collection is deployed on. |
| `cursor` (String) | The cursor to use for pagination. |
| `limit` (Int) | The maximum number of NFT collections to return. |
