---
title: "getLatestPairs"
description: "Returns new tokens listed over the last three days."
---

**Type:** Query

**Returns:** [LatestPairConnection](/api-reference/types/latestpairconnection)

## Arguments for getLatestPairs

| Name | Description |
|------|-------------|
| `limit` (Int) | The maximum number of tokens to return. |
| `networkFilter` ([[Int!]](/api-reference/types/int)) | The list of network IDs to filter by. |
| `exchangeFilter` ([[String!]](/api-reference/types/string)) | The list of exchanges to filter by. |
| `minLiquidityFilter` (Int) | The minimum amount of liquidity to filter by. |
| `cursor` (String) | The cursor to use for pagination. |
