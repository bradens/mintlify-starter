## Arguments for onNftAssetsCreated

| Name | Description |
|------|-------------|
| `address` (String) | The contract address for the NFT collection |
| `networkId` (Int) | The network ID the NFT collection is deployed on. |
| `tokenId` (String) | The token ID of the NFT asset. |
