---
title: "ExplorerTokenData"
description: "Third party token data sourced from off chain."
---

**Type:** [ExplorerTokenData](/api-reference/types/explorertokendata)

## Fields

| Name | Description |
|------|-------------|
| `id` → [String!](/api-reference/types/string) | The ID of the token (`address:networkId`). |
| `blueCheckmark` → Boolean | Whether the token has been verified on CoinGecko. |
| `description` → String | A description of the token. |
| `divisor` → String | The precision to which the token can be divided. |
| `tokenPriceUSD` → String | The token price in USD. |
| `tokenType` → String | The token type. |
