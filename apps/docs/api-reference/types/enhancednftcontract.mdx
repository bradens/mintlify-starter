---
title: "EnhancedNftContract"
description: "Metadata for an NFT collection."
---

**Type:** [EnhancedNftContract](/api-reference/types/enhancednftcontract)

## Fields

| Name | Description |
|------|-------------|
| `id` → [String!](/api-reference/types/string) | The ID of the NFT collection (`address`:`networkId`). |
| `address` → [String!](/api-reference/types/string) | The contract address of the NFT collection. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the NFT collection is deployed on. |
| `ercType` → [String!](/api-reference/types/string) | The token standard. Can be a variation of `ERC-721` or `ERC-1155`. |
| `name` → String | The name of the NFT collection. |
| `description` → String | The description of the NFT collection. |
| `symbol` → String | The symbol of the NFT collection. |
| `totalSupply` → String | The total supply of the NFT collection. |
| `socialLinks` → [SocialLinks](/api-reference/types/sociallinks) | Community gathered links for the socials of this NFT collection. |
| `image` → String | The URL for an image of the NFT collection. |
| `labels` → [[ContractLabel]](/api-reference/types/contractlabel) | A list of labels for the NFT collection. |
