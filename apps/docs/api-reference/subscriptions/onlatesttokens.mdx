---
title: "onLatestTokens"
description: "Subscription operation"
---

**Returns:** [LatestToken](/api-reference/types/latesttoken)

### Arguments

<ResponseField name="id" type="String">
  The ID of the token (`tokenAddress`:`networkId`).
</ResponseField>

<ResponseField name="networkId" type="Int">
  The network ID the token is deployed on.
</ResponseField>

<ResponseField name="tokenAddress" type="String">
  The address of the token contract.
</ResponseField>

### Schema


```graphql 
onLatestTokens(id: String, networkId: Int, tokenAddress: String): LatestToken
```
