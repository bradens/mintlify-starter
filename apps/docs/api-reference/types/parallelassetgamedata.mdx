---
title: "ParallelAssetGameData"
description: "Game data for a Parallel asset."
---

**Type:** [ParallelAssetGameData](/api-reference/types/parallelassetgamedata)

## Fields

| Name | Description |
|------|-------------|
| `rarity` → String | The rarity of the asset. Can be `Common`, `Uncommon`, `Rare`, `Legendary`, or `Prime`. |
| `parallel` → String | The Parallel the asset belongs to. |
| `cost` → String | The energy used to play in-game. |
| `attack` → String | The damage dealt when engaged in combat. |
| `health` → String | The possible damage received before being destroyed. |
| `cardType` → String | The card type. Can be `Effect`, `Relic`, `Unit`, `Upgrade` or `Paragon`. |
| `subtype` → String | The card subtype. Can be `Pirate`, `Vehicle` or `Clone`. |
| `functionText` → String | The description of the card's in-game abilities. |
| `passiveAbility` → String | The description of the card's passive ability. |
