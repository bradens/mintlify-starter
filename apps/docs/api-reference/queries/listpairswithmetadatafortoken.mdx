---
title: "listPairsWithMetadataForToken"
description: "Returns a list of pair metadata for a token."
---

**Type:** Query

**Returns:** [ListPairsForTokenResponse!](/api-reference/types/listpairsfortokenresponse)

## Arguments for listPairsWithMetadataForToken

| Name | Description |
|------|-------------|
| `limit` (Int) | The maximum number of pairs to return. |
| `networkId` ([Int!](/api-reference/types/int)) | The network ID the token is deployed on. |
| `tokenAddress` ([String!](/api-reference/types/string)) | The contract address of the token. |
