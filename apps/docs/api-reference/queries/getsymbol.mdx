---
title: "getSymbol"
description: "Returns charting metadata for a given pair. Used for implementing a Trading View datafeed."
---

**Type:** Query

**Returns:** [SymbolResponse](/api-reference/types/symbolresponse)

### Arguments

<ResponseField name="symbol" type="String!" required>
  The ID of the pair (`address:networkId`).
</ResponseField>

<ResponseField name="currencyCode" type="String">
  The currency to use for the response. Can be `USD` (default) or `TOKEN`.
</ResponseField>
