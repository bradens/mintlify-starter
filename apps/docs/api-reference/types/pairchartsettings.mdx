---
title: "PairChartSettings"
description: "Input options for the chart."
---

**Type:** [PairChartSettings](/api-reference/types/pairchartsettings)

## Fields

| Name | Description |
|------|-------------|
| `pairAddress` → [String!](/api-reference/types/string) | The contract address of the pair. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the pair is deployed on. |
| `from` → Int | The unix timestamp for the start of the requested range. |
| `to` → Int | The unix timestamp for the end of the requested range. |
| `quoteToken` → [QuoteToken](/api-reference/types/quotetoken) | The token of interest within the token's top pair. Can be `token0` or `token1`. |
| `resolution` → String | The time frame for each candle. Available options are `1S`, `5S`, `15S`, `30S`, `1`, `5`, `15`, `30`, `60`, `240`, `720`, `1D`, `7D`. |
| `theme` → [ChartTheme](/api-reference/types/charttheme) | The color theme of the chart. |
