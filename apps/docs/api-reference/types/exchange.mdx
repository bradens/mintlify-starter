---
title: "Exchange"
description: "Metadata for a decentralized exchange."
---

**Type:** [Exchange](/api-reference/types/exchange)

## Fields

| Name | Description |
|------|-------------|
| `address` → [String!](/api-reference/types/string) | The contract address of the exchange. |
| `color` → String | The hex string for the exchange color. |
| `id` → [String!](/api-reference/types/string) | The ID of the exchange (`address:id`). |
| `name` → String | The name of the exchange. |
| `exchangeVersion` → String | The version of the exchange, if applicable. |
| `iconUrl` → String | The exchange logo URL. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the exchange is deployed on. |
| `tradeUrl` → String | The URL for the exchange. |
