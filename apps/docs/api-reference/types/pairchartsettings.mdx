### Fields

<ResponseField name="pairAddress" type="String!" required>
  The contract address of the pair.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the pair is deployed on.
</ResponseField>

<ResponseField name="from" type="Int">
  The unix timestamp for the start of the requested range.
</ResponseField>

<ResponseField name="to" type="Int">
  The unix timestamp for the end of the requested range.
</ResponseField>

<a href="/api-reference/types/quotetoken">
  <ResponseField name="quoteToken" type="QuoteToken">
    The token of interest within the token's top pair. Can be `token0` or `token1`.
  </ResponseField>
</a>

<ResponseField name="resolution" type="String">
  The time frame for each candle. Available options are `1S`, `5S`, `15S`, `30S`, `1`, `5`, `15`, `30`, `60`, `240`, `720`, `1D`, `7D`.
</ResponseField>

<a href="/api-reference/types/charttheme">
  <ResponseField name="theme" type="ChartTheme">
    The color theme of the chart.
  </ResponseField>
</a>
