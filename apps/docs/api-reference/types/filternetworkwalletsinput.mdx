---
title: "FilterNetworkWalletsInput"
description: "The input for filtering wallets for a network."
---

**Type:** [FilterNetworkWalletsInput](/api-reference/types/filternetworkwalletsinput)

## Fields

| Name | Description |
|------|-------------|
| `networkId` → [Int!](/api-reference/types/int) | The network ID to filter wallets for |
| `filters` → [WalletNetworkFilters](/api-reference/types/walletnetworkfilters) | A set of filters to apply. |
| `wallets` → [[String]](/api-reference/types/string) | A list of wallet addresses to filter by. |
| `excludeLabels` → [[String]](/api-reference/types/string) | Exclude wallets with these labels. |
| `includeLabels` → [[String]](/api-reference/types/string) | Include wallets with these labels. |
| `rankings` → [[WalletNetworkRanking]](/api-reference/types/walletnetworkranking) | A list of ranking attributes to apply. |
| `limit` → Int | The maximum number of wallets to return. |
| `offset` → Int | Where in the list the server should start when returning items. Use `count`+`offset` from the previous query to request the next page of results. |
