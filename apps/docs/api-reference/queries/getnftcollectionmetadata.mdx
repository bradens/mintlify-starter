---
title: "getNftCollectionMetadata"
description: "Returns stats for an NFT collection across different time frames."
---

**Type:** Query

**Returns:** [NftCollectionMetadataResponse](/api-reference/types/nftcollectionmetadataresponse)

## Arguments for getNftCollectionMetadata

| Name | Description |
|------|-------------|
| `collectionId` (String) | The ID of the NFT collection (`address`:`networkId`). For example, `0xbc4ca0eda7647a8ab7c2061c2e118a18a936f13d:1`. |
| `address` (String) | The contract address of the NFT collection. |
| `networkId` (Int) | The network ID the NFT collection is deployed on. |
