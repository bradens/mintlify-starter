---
title: "onNftPoolEventsCreated"
description: "Live streamed NFT pool events for a given pool address or collection address."
---

**Type:** Subscription

**Returns:** [AddNftPoolEventsOutput](/api-reference/types/addnftpooleventsoutput)

## Arguments for onNftPoolEventsCreated

| Name | Description |
|------|-------------|
| `poolAddress` (String) | The NFT pool address to filter by. |
| `networkId` (Int) | The network ID to filter by. |
| `collectionAddress` (String) | The contract address of the NFT collection to filter by. |
| `exchangeAddress` (String) | The contract address of the NFT AMM marketplace to filter by. |
