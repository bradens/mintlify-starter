---
title: "liquidityMetadataByToken"
description: "Returns liquidity metadata for a given token. Includes liquidity lock data for up to 100 pairs that the token is in."
---

**Type:** Query

**Returns:** [LiquidityMetadataByToken!](/api-reference/types/liquiditymetadatabytoken)

## Arguments for liquidityMetadataByToken

| Name | Description |
|------|-------------|
| `tokenAddress` ([String!](/api-reference/types/string)) | The address of the token. |
| `networkId` ([Int!](/api-reference/types/int)) | The network id of the token. |
