---
title: "getNftPoolStats"
description: "Returns aggregated NFT pool/collection stats for a given time frame."
---

**Returns:** [NftPoolStatsResponse](/api-reference/types/nftpoolstatsresponse)

### Arguments

<ResponseField name="collectionAddress" type="String!" required>
  The contract address of the NFT collection.
</ResponseField>

<ResponseField name="exchangeAddress" type="String!" required>
  The NFT AMM marketplace address to filter by.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the NFT collection is deployed on.
</ResponseField>

<ResponseField name="startTime" type="Int!" required>
  The unix timestamp for the start of the requested range.
</ResponseField>

<ResponseField name="endTime" type="Int!" required>
  The unix timestamp for the end of the requested range.
</ResponseField>

<ResponseField name="poolAddress" type="String">
  The NFT pool address to filter by.
</ResponseField>

### Schema


```graphql 
"""Returns aggregated NFT pool/collection stats for a given time frame."""
getNftPoolStats(collectionAddress: String!, exchangeAddress: String!, networkId: Int!, startTime: Int!, endTime: Int!, poolAddress: String): NftPoolStatsResponse
```
