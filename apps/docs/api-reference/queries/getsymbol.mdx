---
title: "getSymbol"
description: "Returns charting metadata for a given pair. Used for implementing a Trading View datafeed."
---

**Type:** Query

**Returns:** [SymbolResponse](/api-reference/types/symbolresponse)

## Arguments for getSymbol

| Name | Description |
|------|-------------|
| `symbol` ([String!](/api-reference/types/string)) | The ID of the pair (`address:networkId`). |
| `currencyCode` (String) | The currency to use for the response. Can be `USD` (default) or `TOKEN`. |
