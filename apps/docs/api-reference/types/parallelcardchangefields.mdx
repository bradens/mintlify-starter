### Fields

<ResponseField name="parallelId" type="String">
  The ID used to match other cards with the same name but different class.
</ResponseField>

<ResponseField name="rarity" type="String">
  The rarity of the asset. Can be `Common`, `Uncommon`, `Rare`, `Legendary`, or `Prime`.
</ResponseField>

<ResponseField name="class" type="String">
  The card class. Can be `Art Card`, `Asset`, `Card Back`, `FE`, `Masterpiece`, `PL`, or `SE`.
</ResponseField>

<ResponseField name="supply" type="String">
  The total supply of this individual asset.
</ResponseField>

<ResponseField name="parallel" type="String">
  The Parallel the asset belongs to.
</ResponseField>

<ResponseField name="cost" type="String">
  The energy used to play in-game.
</ResponseField>

<ResponseField name="attack" type="String">
  The damage dealt when engaged in combat.
</ResponseField>

<ResponseField name="health" type="String">
  The possible damage received before being destroyed.
</ResponseField>

<ResponseField name="cardType" type="String">
  The card type. Can be `Effect`, `Relic`, `Unit`, `Upgrade` or `Paragon`.
</ResponseField>

<ResponseField name="subtype" type="String">
  The card subtype. Can be `Pirate`, `Vehicle` or `Clone`.
</ResponseField>

<ResponseField name="functionText" type="String">
  The description of the card's in-game abilities.
</ResponseField>

<ResponseField name="passiveAbility" type="String">
  The description of the card's passive ability.
</ResponseField>

<ResponseField name="flavourText" type="String">
  The asset description, sourced off-chain. Usually equal to the asset's on-chain `description`.
</ResponseField>

<ResponseField name="expansion" type="String">
  The expansion used for naming base and expansion sets.
</ResponseField>

<ResponseField name="paraset" type="String">
  The paraset the asset belongs to.
</ResponseField>

<ResponseField name="artist" type="String">
  The artist name.
</ResponseField>
