title: "NftEventTradeItemBase"
description: "Fields that are common in all items offered or received as part of an nft trade."
---

**Type:** [NftEventTradeItemBase](/api-reference/types/nfteventtradeitembase)

## Fields

| Name | Description |
|------|-------------|
| `address` → [String!](/api-reference/types/string) | The contract address for the item. |
| `amount` → String | The number of items transferred. |
| `recipient` → String | The recipient of the items. |
| `type` → [NftEventTradeItemType!](/api-reference/types/nfteventtradeitemtype) | The type of item involved in the trade. (NFT or TOKEN) |
