---
title: "listTopTokens"
description: "Returns a list of trending tokens across any given network(s)."
---

**Type:** Query

**Returns:** [[TokenWithMetadata!]](/api-reference/types/tokenwithmetadata)

## Arguments for listTopTokens

| Name | Description |
|------|-------------|
| `limit` (Int) | The maximum number of tokens to return. Max is `50`. |
| `networkFilter` ([[Int!]](/api-reference/types/int)) | The list of network IDs to filter by. |
| `resolution` (String) | The time frame for trending results. Can be `1`, `5`, `15`, `30`, `60`, `240`, `720`, or `1D`. |
