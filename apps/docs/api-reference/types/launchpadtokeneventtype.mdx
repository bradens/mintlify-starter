---
title: "LaunchpadTokenEventType"
description: "The type of event. Note that associated statistics such as `buyCount1`, `price`, etc. are only available for `Updated` events."
---

**Type:** [LaunchpadTokenEventType](/api-reference/types/launchpadtokeneventtype)

## Values

| Value | Description |
|-------|-------------|
| `Deployed` | The token has been discovered |
| `Created` | The token has been created with metadata |
| `Updated` | The token&#39;s statistics have been updated |
| `Completed` | The token has been completed |
| `Migrated` | The token has been migrated |
