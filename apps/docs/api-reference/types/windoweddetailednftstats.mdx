**Type:** [WindowedDetailedNftStats](/api-reference/types/windoweddetailednftstats)

## Fields

| Name | Description |
|------|-------------|
| `duration` → [DetailedNftStatsDuration!](/api-reference/types/detailednftstatsduration) | The duration used to request detailed NFT stats. |
| `start` → [Int!](/api-reference/types/int) | The unix timestamp for the start of the window. |
| `end` → [Int!](/api-reference/types/int) | The unix timestamp for the end of the window. |
| `timestamps` → [[DetailedNftStatsBucketTimestamp]!](/api-reference/types/detailednftstatsbuckettimestamp) | The list of start/end timestamps broken down for each bucket within the window. |
| `statsUsd` → [WindowedDetailedNftCurrencyStats!](/api-reference/types/windoweddetailednftcurrencystats) | The currency stats in USD, such as volume. |
| `statsNetworkBaseToken` → [WindowedDetailedNftCurrencyStats!](/api-reference/types/windoweddetailednftcurrencystats) | The currency stats in the network's base token, such as volume. |
| `statsNonCurrency` → [WindowedDetailedNftNonCurrencyStats!](/api-reference/types/windoweddetailednftnoncurrencystats) | The numerical stats, such as number of buyers. |
