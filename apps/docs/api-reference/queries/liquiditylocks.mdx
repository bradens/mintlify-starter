---
title: "liquidityLocks"
description: "Returns liquidity locks for a given pair."
---

**Returns:** [LiquidityLockConnection](/api-reference/types#liquiditylockconnection)

### Arguments

<ResponseField name="pairAddress" type="String">
  The address of the pair.
</ResponseField>

<ResponseField name="tokenAddress" type="String">
  The address of the token.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network id of the pair.
</ResponseField>

<ResponseField name="cursor" type="String">
  A cursor for use in pagination.
</ResponseField>

### Schema


```graphql 
"""Returns liquidity locks for a given pair."""
liquidityLocks(pairAddress: String, tokenAddress: String, networkId: Int!, cursor: String): LiquidityLockConnection
```
