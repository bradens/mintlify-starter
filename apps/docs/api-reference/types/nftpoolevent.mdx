---
title: "NftPoolEvent"
description: "An NFT pool transaction."
---

**Type:** [NftPoolEvent](/api-reference/types/nftpoolevent)

## Fields

| Name | Description |
|------|-------------|
| `id` → [String!](/api-reference/types/string) | The ID of the NFT pool (`poolAddress`:`networkId`). For example, `0xdbea289dcc10eed8431e78753414a3d81b8e7201:1`. |
| `poolAddress` → [String!](/api-reference/types/string) | The contract address of the NFT pool. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the NFT collection is deployed on. |
| `collectionAddress` → [String!](/api-reference/types/string) | The contract address of the NFT collection. |
| `collectionId` → [String!](/api-reference/types/string) | The ID of the NFT collection (`collectionAddress`:`networkId`). |
| `exchangeAddress` → [String!](/api-reference/types/string) | The contract address of the NFT AMM marketplace. |
| `tokenAddress` → [String!](/api-reference/types/string) | The contract address of the liquidity token of the pool (usually WETH). |
| `blockNumber` → [Int!](/api-reference/types/int) | The block number for the transaction. |
| `logIndex` → [Int!](/api-reference/types/int) | The index of the log in the block. |
| `transactionIndex` → [Int!](/api-reference/types/int) | The index of the transaction within the block. |
| `blockHash` → [String!](/api-reference/types/string) | The hash of the block where the transaction occurred. |
| `transactionHash` → [String!](/api-reference/types/string) | The unique hash for the transaction. |
| `timestamp` → [Int!](/api-reference/types/int) | The unix timestamp for the transaction. |
| `maker` → [String!](/api-reference/types/string) | The wallet address that transacted. |
| `poolType` → [NftPoolType!](/api-reference/types/nftpooltype) | The type of liquidity pool. |
| `eventType` → [NftPoolEventType!](/api-reference/types/nftpooleventtype) | The event type of the transaction. |
| `data` → [NftPoolEventData!](/api-reference/types/nftpooleventdata) | The event-specific data for the transaction. |
