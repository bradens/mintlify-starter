**Type:** Query

## Arguments for getLatestTokens

| Name | Description |
|------|-------------|
| `limit` (Int) | The maximum number of tokens to return. Maximum number of tokens is 50. |
| `networkFilter` ([[Int!]](/api-reference/types/int)) | The list of network IDs to filter by. |
| `offset` (Int) | The offset to use for pagination. |
