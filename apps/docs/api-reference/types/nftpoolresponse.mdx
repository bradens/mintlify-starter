---
title: "NftPoolResponse"
description: "An NFT pool."
---

**Type:** [NftPoolResponse](/api-reference/types/nftpoolresponse)

## Fields

| Name | Description |
|------|-------------|
| `poolId` → [String!](/api-reference/types/string) | The ID of the NFT pool (`poolAddress`:`networkId`). For example, `0xdbea289dcc10eed8431e78753414a3d81b8e7201:1`. |
| `tokenAddress` → [String!](/api-reference/types/string) | The contract address of the liquidity token of the pool (usually WETH). |
| `collectionAddress` → [String!](/api-reference/types/string) | The contract address of the NFT collection. |
| `collectionName` → [String!](/api-reference/types/string) | The contract name of the NFT collection. |
| `collectionSymbol` → String | The symbol of the NFT collection. |
| `exchangeAddress` → [String!](/api-reference/types/string) | The contract address of the NFT AMM marketplace. |
| `poolAddress` → [String!](/api-reference/types/string) | The contract address of the NFT pool. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the NFT collection is deployed on. |
| `poolType` → [NftPoolType!](/api-reference/types/nftpooltype) | The type of liquidity pool. |
| `poolVariant` → [GraphQlNftPoolVariant!](/api-reference/types/graphqlnftpoolvariant) | The pool variant. Can be `ERC20` or `NATIVE`. |
| `bondingCurveType` → [BondingCurveType!](/api-reference/types/bondingcurvetype) | The bonding curve type that defines how the prices of NFTs change after each buy or sell within a pool. |
| `bondingCurveAddress` → [String!](/api-reference/types/string) | The contract address of the bonding curve. |
| `assetRecipientAddress` → [String!](/api-reference/types/string) | The wallet address that will receive the tokens or NFT sent to the pair during swaps. |
| `delta` → [String!](/api-reference/types/string) | The current delta used in the bonding curve. |
| `owner` → [String!](/api-reference/types/string) | The wallet address of the pool owner. |
| `fee` → [String!](/api-reference/types/string) | The current fee for pool. |
| `poolFeesAllTimeT` → String | No description provided |
| `poolFeesAllTimeNBT` → String | No description provided |
| `floorNBT` → String | The current price at which the pool is willing to sell an NFT in the network&#39;s base token. Only applicable for `SELL` and `BUY_AND_SELL` pool types. |
| `offerNBT` → String | The current price at which the pool is willing to buy an NFT in the network&#39;s base token. Only applicable for `BUY` and `BUY_AND_SELL` pool types. |
| `spotPriceNBT` → [String!](/api-reference/types/string) | The spot price in the network&#39;s base token. |
| `balanceNBT` → [String!](/api-reference/types/string) | The current pool liquidity in the network&#39;s base token. |
| `nftBalance` → Int | The current number of NFTs in the pool. |
| `nftBalanceV2` → [String!](/api-reference/types/string) | The current number of NFTs in the pool. |
| `volumeAllTimeNBT` → [String!](/api-reference/types/string) | The total volume of the pool in the network&#39;s base token over the pool&#39;s lifetime. |
| `floorT` → String | The current price at which the pool is willing to sell an NFT in the pool&#39;s liquidity token. Only applicable for `SELL` and `BUY_AND_SELL` pool types. |
| `offerT` → String | The current price at which the pool is willing to buy an NFT in the pool&#39;s liquidity token. Only applicable for `BUY` and `BUY_AND_SELL` pool types. |
| `spotPriceT` → [String!](/api-reference/types/string) | The instantaneous price for selling 1 NFT to the pool in the pool&#39;s liquidity token. |
| `balanceT` → [String!](/api-reference/types/string) | The current pool liquidity in the pool&#39;s liquidity token. |
| `volumeAllTimeT` → [String!](/api-reference/types/string) | The total volume of the pool in the pool&#39;s liquidity token over the pool&#39;s lifetime. |
| `nftVolumeAllTime` → [Int!](/api-reference/types/int) | The total number of NFTs bought and sold over the pool&#39;s lifetime. |
| `nftVolumeAllTimeV2` → [String!](/api-reference/types/string) | The total number of NFTs bought and sold over the pool&#39;s lifetime. |
| `nftAssets` → [[NftAsset]](/api-reference/types/nftasset) | The list of NFT assets in the pool. |
| `poolNftType` → [PoolNftType](/api-reference/types/poolnfttype) | The type of NFT in the pool. |
| `royalties` → [[NftPoolRoyalty]](/api-reference/types/nftpoolroyalty) | The list of royalties for the pool. Only applicable for `SUDOSWAP_V2` pools. |
| `propertyChecker` → String | The property checker contract address for the pool. |
| `version` → [NftPoolContractVersion](/api-reference/types/nftpoolcontractversion) | The NFT pool contract version. Can be `SUDOSWAP_V1` or `SUDOSWAP_V2`. |
| `acceptedNftTokenIds` → [[String]](/api-reference/types/string) | For ERC1155 pools, the list of NFT token IDs that are accepted by the pool. |
