---
title: "listPairsForToken"
description: "Returns a list of pairs containing a given token."
---

**Type:** Query

**Returns:** [[Pair]!](/api-reference/types/pair)

### Arguments

<ResponseField name="limit" type="Int">
  The maximum number of pairs to return.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the token is deployed on.
</ResponseField>

<ResponseField name="tokenAddress" type="String!" required>
  The contract address of the token.
</ResponseField>
