| `toOrFrom` → [StringEqualsCondition](/api-reference/types/stringequalscondition) | Trigger the webhook if either the to or the from address matches. |
| `input` → [StringContainsCondition](/api-reference/types/stringcontainscondition) | Trigger the webhook if the  contains or doesn&#39;t contain the specified string. |
| `ignoreTokenPairEvents` → Boolean | Do not trigger the webhook if the raw transaction is handled by the TokenPairEvent webhook. |
| `ignoreNftEvents` → Boolean | Do not trigger the webhook if the raw transaction is handled by the NftEvent webhook. |
