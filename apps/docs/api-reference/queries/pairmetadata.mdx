---
title: "pairMetadata"
description: "Returns metadata for a pair of tokens."
---

**Returns:** [PairMetadata!](/api-reference/types#pairmetadata)

### Arguments

<ResponseField name="pairId" type="String!" required>
  The ID of the pair (`address:networkId`).
</ResponseField>

<a href="/api-reference/enums#quotetoken">
  <ResponseField name="quoteToken" type="QuoteToken">
    The token of interest within the pair. Can be `token0` or `token1`.
  </ResponseField>
</a>

<a href="/api-reference/enums#tokenpairstatisticstype">
  <ResponseField name="statsType" type="TokenPairStatisticsType">
    The type of statistics returned. Can be `FILTERED` or `UNFILTERED`
  </ResponseField>
</a>

### Schema


```graphql 
"""Returns metadata for a pair of tokens."""
pairMetadata(pairId: String!, quoteToken: QuoteToken, statsType: TokenPairStatisticsType): PairMetadata!
```
