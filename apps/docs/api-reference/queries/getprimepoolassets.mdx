---
title: "getPrimePoolAssets"
description: "Returns a list of Prime pool cached assets."
keywords: ['getPrimePoolAssets']
---

**Returns:** [PrimePoolAssetConnection](/api-reference/types/primepoolassetconnection)

### Arguments

<ResponseField name="networkId" type="Int!" required>
  The network ID the Prime pool is deployed on.
</ResponseField>

<ResponseField name="poolContractAddress" type="String">
  The contract address of the Prime pool.
</ResponseField>

<ResponseField name="poolId" type="String">
  The pool ID for the Prime pool, within the contract.
</ResponseField>

<ResponseField name="walletAddress" type="String">
  The owner wallet address to filter assets by.
</ResponseField>

<ResponseField name="limit" type="Int">
  The maximum number of Prime Pool assets to return.
</ResponseField>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>

### Schema


```graphql 
"""Returns a list of Prime pool cached assets."""
getPrimePoolAssets(networkId: Int!, poolContractAddress: String, poolId: String, walletAddress: String, limit: Int, cursor: String): PrimePoolAssetConnection
```
