---
title: "listTopTokens"
description: "Returns a list of trending tokens across any given network(s)."
---

<Warning>
  This query is deprecated. This query is no longer supported and will not return up to date data. Use `filterTokens` instead.
</Warning>

**Returns:** [[TokenWithMetadata!]](/api-reference/types#tokenwithmetadata)

### Arguments

<ResponseField name="limit" type="Int">
  The maximum number of tokens to return. Max is `50`.
</ResponseField>

<ResponseField name="networkFilter" type="[Int!]">
  The list of network IDs to filter by.
</ResponseField>

<ResponseField name="resolution" type="String">
  The time frame for trending results. Can be `1`, `5`, `15`, `30`, `60`, `240`, `720`, or `1D`.
</ResponseField>

### Schema


```graphql 
"""Returns a list of trending tokens across any given network(s)."""
listTopTokens(limit: Int, networkFilter: [Int!], resolution: String): [TokenWithMetadata!] @deprecated(reason: "This query is no longer supported and will not return up to date data. Use `filterTokens` instead.")
```
