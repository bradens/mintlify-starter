---
title: "getLatestTokens"
description: "Returns a list of latest tokens."
---

<Warning>
  This query is deprecated. This query is no longer supported. Use `filterTokens` with a createdAt: DESC filter instead.
</Warning>

**Returns:** [LatestTokenConnection](/api-reference/types#latesttokenconnection)

### Arguments

<ResponseField name="limit" type="Int">
  The maximum number of tokens to return. Maximum number of tokens is 50.
</ResponseField>

<ResponseField name="networkFilter" type="[Int!]">
  The list of network IDs to filter by.
</ResponseField>

<ResponseField name="offset" type="Int">
  The offset to use for pagination.
</ResponseField>

### Schema

"""Returns a list of latest tokens."""
