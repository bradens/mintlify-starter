---
title: "getNftPoolEvents"
description: "Returns transactions for an NFT collection across all NFT pools or within a given pool."
---

**Type:** Query

**Returns:** [NftPoolEventsResponse](/api-reference/types/nftpooleventsresponse)

### Arguments

<ResponseField name="networkId" type="Int!" required>
  The network ID the NFT collection is deployed on.
</ResponseField>

<ResponseField name="collectionAddress" type="String">
  The contract address of the NFT collection.
</ResponseField>

<ResponseField name="poolAddress" type="String">
  The NFT pool address to filter by.
</ResponseField>

<ResponseField name="exchangeAddress" type="String">
  The NFT AMM marketplace address to filter by.
</ResponseField>

<a href="/api-reference/types/nftpooleventtype">
  <ResponseField name="eventTypes" type="[NftPoolEventType!]">
    The event types to filter by.
  </ResponseField>
</a>

<a href="/api-reference/types/eventquerytimestampinput">
  <ResponseField name="timestamp" type="EventQueryTimestampInput">
    The time range to filter by.
  </ResponseField>
</a>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>

<ResponseField name="limit" type="Int">
  The maximum number of NFT pool events to return.
</ResponseField>
