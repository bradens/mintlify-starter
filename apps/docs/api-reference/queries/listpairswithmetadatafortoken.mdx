---
title: "listPairsWithMetadataForToken"
description: "Returns a list of pair metadata for a token."
---

**Returns:** [ListPairsForTokenResponse!](/api-reference/types/listpairsfortokenresponse)

### Arguments

<ResponseField name="limit" type="Int">
  The maximum number of pairs to return.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the token is deployed on.
</ResponseField>

<ResponseField name="tokenAddress" type="String!" required>
  The contract address of the token.
</ResponseField>

### Schema


```graphql 
"""Returns a list of pair metadata for a token."""
listPairsWithMetadataForToken(limit: Int, networkId: Int!, tokenAddress: String!): ListPairsForTokenResponse!
```
