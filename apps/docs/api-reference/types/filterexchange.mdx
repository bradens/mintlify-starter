---
title: "FilterExchange"
description: "Metadata for an exchange."
---

**Type:** [FilterExchange](/api-reference/types/filterexchange)

## Fields

| Name | Description |
|------|-------------|
| `address` → [String!](/api-reference/types/string) | The address for the exchange factory contract. |
| `id` → [String!](/api-reference/types/string) | The ID of the exchange (`address:networkId`). |
| `name` → String | The name of the exchange. |
| `exchangeVersion` → String | The version of the exchange. For example, `3` for UniswapV3. |
| `iconUrl` → String | The URL for the exchange&#39;s icon. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the factory is deployed on. |
| `tradeUrl` → String | The URL for the exchange&#39;s trading platform. |
