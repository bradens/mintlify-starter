---
title: "onEventsCreated"
description: "Live-streamed transactions for a token."
---

**Returns:** [AddEventsOutput](/api-reference/types/addeventsoutput)

### Arguments

<ResponseField name="address" type="String">
  The pair contract address.
</ResponseField>

<ResponseField name="id" type="String">
  The ID of the pair (`address`:`networkId`).
</ResponseField>

<a href="/api-reference/types/quotetoken">
  <ResponseField name="quoteToken" type="QuoteToken">
    The token of interest within the pair. Can be `token0` or `token1`.
  </ResponseField>
</a>

<ResponseField name="networkId" type="Int">
  The networkId to use when getting all events per network
</ResponseField>

### Schema


```graphql 
"""Live-streamed transactions for a token."""
onEventsCreated(address: String, id: String, quoteToken: QuoteToken, networkId: Int): AddEventsOutput
```
