---
title: "pairMetadata"
description: "Returns metadata for a pair of tokens."
---

**Type:** Query

**Returns:** [PairMetadata!](/api-reference/types/pairmetadata)

## Arguments for pairMetadata

| Name | Description |
|------|-------------|
| `pairId` ([String!](/api-reference/types/string)) | The ID of the pair (`address:networkId`). |
| `quoteToken` ([QuoteToken](/api-reference/types/quotetoken)) | The token of interest within the pair. Can be `token0` or `token1`. |
| `statsType` ([TokenPairStatisticsType](/api-reference/types/tokenpairstatisticstype)) | The type of statistics returned. Can be `FILTERED` or `UNFILTERED` |
