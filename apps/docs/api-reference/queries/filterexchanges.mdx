---
title: "filterExchanges"
description: "Returns a list of exchanges based on a variety of filters."
---

**Returns:** [ExchangeFilterConnection](/api-reference/types/exchangefilterconnection)

### Arguments

<a href="/api-reference/types/exchangefilters">
  <ResponseField name="filters" type="ExchangeFilters">
    A set of filters to apply.
  </ResponseField>
</a>

<ResponseField name="phrase" type="String">
  A phrase to search for. Can match an exchange address or ID (`address:networkId`), or partially match an exchange name.
</ResponseField>

<a href="/api-reference/types/exchangeranking">
  <ResponseField name="rankings" type="[ExchangeRanking]">
    A list of ranking attributes to apply.
  </ResponseField>
</a>

<ResponseField name="limit" type="Int">
  The maximum number of exchanges to return.
</ResponseField>

<ResponseField name="offset" type="Int">
  Where in the list the server should start when returning items. Use `count`+`offset` from the previous query to request the next page of results.
</ResponseField>

### Schema


```graphql 
"""Returns a list of exchanges based on a variety of filters."""
filterExchanges(filters: ExchangeFilters, phrase: String, rankings: [ExchangeRanking], limit: Int, offset: Int): ExchangeFilterConnection
```
