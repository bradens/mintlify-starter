---
title: "EnhancedToken"
description: "Metadata for a token."
---

**Type:** [EnhancedToken](/api-reference/types/enhancedtoken)

## Fields

| Name | Description |
|------|-------------|
| `address` → [String!](/api-reference/types/string) | The contract address of the token. |
| `cmcId` → Int | The token ID on CoinMarketCap. |
| `decimals` → [Int!](/api-reference/types/int) | The precision to which the token can be divided. For example, the smallest unit for USDC is 0.000001 (6 decimals). |
| `id` → [String!](/api-reference/types/string) | The ID of the token (`address:networkId`). |
| `isScam` → Boolean | Whether the token has been flagged as a scam. |
| `name` → String | The token name. For example, `ApeCoin`. |
| `networkId` → [Int!](/api-reference/types/int) | The network ID the token is deployed on. |
| `symbol` → String | The token symbol. For example, `APE`. |
| `totalSupply` → String | The total supply of the token. |
| `socialLinks` → [SocialLinks](/api-reference/types/sociallinks) | Community gathered links for the socials of this token. |
| `info` → [TokenInfo](/api-reference/types/tokeninfo) | More metadata about the token. |
| `explorerData` → [ExplorerTokenData](/api-reference/types/explorertokendata) | Information about the token from 3rd party sources. |
| `exchanges` → [[Exchange!]](/api-reference/types/exchange) | A list of exchanges where the token has been traded. |
| `imageThumbUrl` → String | The thumbnail token logo URL. |
| `imageSmallUrl` → String | The small token logo URL. |
| `imageLargeUrl` → String | The large token logo URL. |
| `circulatingSupply` → String | The circulating supply of the token. |
| `pooled` → String | The amount of this token in the pair. |
| `creatorAddress` → String | The token creator's wallet address. |
| `createBlockNumber` → Int | The block height the token was created at. |
| `createTransactionHash` → String | The transaction hash of the token's creation. |
| `createdAt` → Int | The unix timestamp for the creation of the token. |
| `mintable` → String | Whether or not the token is mintable |
| `freezable` → String | Whether or not the token is freezable |
| `launchpad` → [LaunchpadData](/api-reference/types/launchpaddata) | The launchpad data for the token, if applicable. |
