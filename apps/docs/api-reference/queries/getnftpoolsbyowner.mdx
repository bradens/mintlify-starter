---
title: "getNftPoolsByOwner"
description: "Returns a list of NFT pools for a given owner."
---

**Type:** Query

**Returns:** [GetNftPoolsResponse](/api-reference/types/getnftpoolsresponse)

### Arguments

<ResponseField name="ownerAddress" type="String!" required>
  The contract address of the pool owner.
</ResponseField>

<ResponseField name="networkId" type="Int!" required>
  The network ID the NFT collection is deployed on.
</ResponseField>

<ResponseField name="exchangeAddress" type="String">
  The NFT AMM marketplace address to filter by.
</ResponseField>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>

<ResponseField name="limit" type="Int">
  The maximum number of NFT pools to return.
</ResponseField>
