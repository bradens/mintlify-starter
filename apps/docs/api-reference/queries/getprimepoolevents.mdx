---
title: "getPrimePoolEvents"
description: "Returns a list of Prime pool events."
keywords: ['getPrimePoolEvents']
---

**Returns:** [PrimePoolEventConnection](/api-reference/types/primepooleventconnection)

### Arguments

<ResponseField name="networkId" type="Int!" required>
  The network ID the Prime pool is deployed on.
</ResponseField>

<ResponseField name="poolContractAddress" type="String">
  The contract address of the Prime pool.
</ResponseField>

<ResponseField name="poolId" type="String">
  The pool ID for the Prime pool, within the contract.
</ResponseField>

<ResponseField name="walletAddress" type="String">
  The calling wallet address to filter events by.
</ResponseField>

<a href="/api-reference/types/primepooleventtype">
  <ResponseField name="eventTypes" type="[PrimePoolEventType]">
    The event types to query for.
  </ResponseField>
</a>

<ResponseField name="limit" type="Int">
  The maximum number of Prime Pool events to return.
</ResponseField>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>

### Schema


```graphql 
"""Returns a list of Prime pool events."""
getPrimePoolEvents(networkId: Int!, poolContractAddress: String, poolId: String, walletAddress: String, eventTypes: [PrimePoolEventType], limit: Int, cursor: String): PrimePoolEventConnection
```
