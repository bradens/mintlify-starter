---
title: "getEventLabels"
description: "Returns a list of event labels for a pair."
---

**Type:** Query

**Returns:** [EventLabelConnection](/api-reference/types/eventlabelconnection)

### Arguments

<ResponseField name="id" type="String!" required>
  The ID of the pair (`address:networkId`).
</ResponseField>

<ResponseField name="limit" type="Int">
  The maximum number of event labels to return.
</ResponseField>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>

<a href="/api-reference/types/rankingdirection">
  <ResponseField name="direction" type="RankingDirection">
    The order to receive the token event labels. Can be `DESC` (default) or `ASC`.
  </ResponseField>
</a>
