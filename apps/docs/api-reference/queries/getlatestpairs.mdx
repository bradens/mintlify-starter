---
title: "getLatestPairs"
description: "Returns new tokens listed over the last three days."
---

<Warning>
  This query is deprecated. This query is longer supported. Instead use filterPairs with sort order on createdAt DESC
</Warning>

**Returns:** [LatestPairConnection](/api-reference/types/latestpairconnection)

### Arguments

<ResponseField name="limit" type="Int">
  The maximum number of tokens to return.
</ResponseField>

<ResponseField name="networkFilter" type="[Int!]">
  The list of network IDs to filter by.
</ResponseField>

<ResponseField name="exchangeFilter" type="[String!]">
  The list of exchanges to filter by.
</ResponseField>

<ResponseField name="minLiquidityFilter" type="Int">
  The minimum amount of liquidity to filter by.
</ResponseField>

<ResponseField name="cursor" type="String">
  The cursor to use for pagination.
</ResponseField>
